# 📊 Diwali Sales Data Analysis

## 📌 Overview  
This project analyzes Diwali sales data to understand customer behavior, sales trends, and business insights.

---

## 🎯 Objectives  
- Analyze customer purchase behavior  
- Identify top-selling products  
- Understand sales trends  
- Generate business insights  

---

## 🛠️ Tools & Technologies  
- Python (Pandas, NumPy, Matplotlib, Seaborn)  
- Jupyter Notebook  
- Excel  

---

## 📂 Project Files  
- Diwali Sales Data.csv → Dataset  
- Diwali_Sales_Analysis.ipynb → Analysis  

---

## 📊 Key Insights  
- Highest spending customers identified  
- Top product categories found  
- Sales analysis by gender, age, and state  

---

## 📈 Conclusion  
Helps businesses improve sales strategies during festivals.
